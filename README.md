# Asyncrum Backend

## Conventions

### ✅ Commit Message
- `[feature] 🌟 새로운 기능 추가`
- `[fix] ✨ 버그 수정`
- `[docs] 🔥 문서 수정`
- `[test] 💥 테스트 코드 추가`
- `[refactor] 🌀 코드 리팩토링`
- `[style] 🔧 코드 의미에 영향을 주지 않는 변경사항`
- `[chore] 🧹 빌드 부분 혹은 패키지 매니저 수정사항`
- `[release] 🎁 새로운 버전을 출시`
